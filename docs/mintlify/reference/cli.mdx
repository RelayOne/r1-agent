R1 ships the `r1` binary today. Companion assets that still carry
legacy names, such as `.stoke/` and `stoke-acp`, are called out
explicitly where relevant.

All subcommands support `--help` to print flags.

## Control plane (sessionctl)

Over the Unix socket, 8 verbs:

| Command | Purpose |
|---|---|
| `r1 pause <id>` | Pause at next safe boundary |
| `r1 resume <id>` | Resume a paused session |
| `r1 approve <id>` | Approve a pending HITL gate |
| `r1 override <id>` | Override a soft-pass or descent verdict |
| `r1 budget <id>` | Adjust session budget |
| `r1 inject <id>` | Inject a message into the session |
| `r1 takeover <id>` | Hand control over to another client |
| `r1 status <id>` | Query session status |
