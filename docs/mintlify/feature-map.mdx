Status legend:

- **Done** — shipped in the current trunk, passing build / test /
  vet, race-clean, integrated into the default execution path.
- **Flagged** — shipped but behind an env var or CLI flag.
- **Ready** — spec at STATUS: ready, awaiting build.
- **Scoped** — spec in flight or under review.
- **Horizon** — on the roadmap, not yet scoped.
