R1 is free open-source software. No license keys, no feature
gates, no telemetry, no phone-home.

## Homebrew (macOS + Linux)

Published by goreleaser on each tag.

```bash
# Canonical tap.
brew install RelayOne/r1-agent/r1
```

## One-line installer

Detects your platform, verifies the cosign signature (keyless
OIDC via sigstore) when cosign is on PATH, and falls back to
building from source if no prebuilt binary exists for your target.

```bash
curl -fsSL https://raw.githubusercontent.com/RelayOne/r1-agent/main/install.sh | bash
```

## Docker

linux/amd64 + linux/arm64; distroless; multi-stage.

```bash
docker pull ghcr.io/RelayOne/r1:latest
```

## From source

Requires Go 1.25+ (1.25 minimum); CGO enabled for SQLite.

```bash
go build ./cmd/r1
go build ./cmd/r1-acp        # Agent Client Protocol adapter
sudo mv r1 stoke-acp /usr/local/bin/
```

## Verify a signed release tarball

Cosign keyless OIDC pinned to the canonical `RelayOne/r1-agent`
release workflow.

```bash
cosign verify-blob \
  --certificate-identity-regexp 'https://github\.com/RelayOne/r1-agent/\.github/workflows/release\.yml@refs/tags/.*' \
  --certificate-oidc-issuer https://token.actions.githubusercontent.com \
  --signature r1_<ver>_<os>_<arch>.tar.gz.sig \
  r1_<ver>_<os>_<arch>.tar.gz
```

<Note>
R1 ships as the `r1` binary today. Legacy storage and companion
surfaces such as `.stoke/` and `stoke-acp` keep their existing names
where compatibility still matters.
</Note>
