# cloudswarm-flagships

Bundled flagship-flow runtimes for the CloudSwarm integration layer.

Current contents:

- `betbuddies_group_runtime` — deterministic runtime contract for the
  BetBuddies private pool flagship flow.
- `dentist_outreach_runtime` — deterministic runtime contract for the
  dentist-outreach / sales-outreach flagship flow.
- `invoice_processor_runtime` — deterministic runtime contract for the
  invoice-ingestion / email-reconciliation flagship flow.
